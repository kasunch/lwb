Start Glossy and stall all other application tasks. 


\begin{DoxyParams}{Parameters}
{\em data\_\-} & A pointer to the flooding data.\\
\hline
\end{DoxyParams}
At the initiator, Glossy reads from the given memory location data provided by the application.

At a receiver, Glossy writes to the given memory location data for the application.

\hline
